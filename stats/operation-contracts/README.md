# Operation contracts source family

This source family cross-routes active mechanic parts whose stats contract is
real but whose result is not a public `summary_surface_catalog` read model.

It currently covers bounded Agon registries, Experience observation
contracts, the via-negativa observation checklist, and Titan memory/runtime
observation contracts. Their active payload stays part-local; this branch
only makes the stats meaning, authority ceiling, and reciprocal route
discoverable.

The authoritative route list is
[`stats/source_home.manifest.json`](../source_home.manifest.json). The reciprocal
part inventory is [`mechanics/topology.json`](../../mechanics/topology.json).
