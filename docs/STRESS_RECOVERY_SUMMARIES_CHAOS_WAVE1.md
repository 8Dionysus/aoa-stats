# Stress Recovery Summaries Chaos Wave 1

`aoa-stats` should summarize this wave only after owner-local receipts, gates,
hints, and bridge candidates exist.

## Good reads

- repeated-window movement
- evidence continuity
- containment quality
- re-entry quality
- regrounding effectiveness
- operator burden
- trust calibration

## Bad reads

- synthetic health truth
- route authority
- proof verdicts
- global "system got smarter" claims

## Integration note

One compact summary example is enough for wave 1.
Do not create a dashboard empire.
