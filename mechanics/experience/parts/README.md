# Experience part map

Choose the smallest part that owns the signal family:

- `micro-friction-receipts/` for friction receipt, recurrence, and inbox shape;
- `adoption-federation-harvest/` for adoption and cross-owner pattern movement;
- `governance-runtime-signals/` for bounded governance and queue readouts;
- `release-watch-office-health/` for certification, watch, rollback, office,
  installation, and release-train health.

Cross-part reuse must be a reference, not a duplicate payload.
