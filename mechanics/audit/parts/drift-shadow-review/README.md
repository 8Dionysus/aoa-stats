# drift-shadow-review

Projects one bounded drift-review summary from the committed cadence-example
chain without replacing source review.

This part owns its operation route and the localized payload roots declared in
`mechanics/topology.json`. Its stats source family owns the bounded meaning;
public schemas and generated outputs stay at their declared publication paths
when consumers depend on those paths.

- source profile: `stats/read-models/active/drift_review_summary.profile.json`
- guide: `docs/DRIFT_REVIEW_SUMMARY.md`
- public schema: `schemas/drift-review-summary.schema.json`
- committed output: `generated/drift_review_summary.min.json`
- shared filesystem-free cadence core:
  `src/aoa_stats_builder/rollout_cadence.py`
- committed-example adapter: `src/aoa_stats_builder/rollout_cadence_sources.py`
- root compatibility facade: `scripts/build_views.py`
- focused proof: `tests/test_drift_review_projection.py`

Release Support owns the campaign projection and the distinct trusted-history
drift readout. This Audit part owns only the explicit cadence review projection.
