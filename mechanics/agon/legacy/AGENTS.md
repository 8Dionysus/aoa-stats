# AGENTS.md

## Agon legacy guide

This directory preserves historical landing notes and closed quests after the
active Agon payload moved into parts.

- Read `INDEX.md` before opening `raw/`.
- Treat `raw/` as historical evidence, not current route law.
- Do not add active configs, schemas, scripts, tests, or generated artifacts.
- Update `former-routes.json` when a former route or active replacement changes.
- Preserve raw files byte-for-byte unless correcting the historical artifact
  itself is the explicit task.
