# titan mechanic

`mechanics/titan/` is the active aoa-stats observation package for the shared
owner mechanic at `aoa-agents/mechanics/titan`.

Publishes bounded Titan incarnation, summon, runtime, memory, and bridge
observations while Titan owners retain identity and gate truth. The
incarnation/summon part deliberately distinguishes a coherent committed roster
projection from a no-observed-ledger compatibility baseline.

Start with `PARTS.md`, then use the selected part's `README.md`,
`CONTRACT.md`, and `VALIDATION.md`.
