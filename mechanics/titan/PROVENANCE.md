# titan provenance

Titan is not a common-center mechanic in `Agents-of-Abyss`. Its shared owner
route is `aoa-agents/mechanics/titan`, where Titan roles, summon boundaries,
runtime roster, service cohorts, and identity-bearing doctrine remain
authoritative. The payload here was regrouped from aoa-stats root districts by
observation, preserving content and stable public publication paths.

The topology records current localized and retained public routes only;
former root placement remains recoverable from Git rename history.
