# session-growth-branch

Summarizes reviewed follow-through hints on `aoa-skills`
`decision_fork_receipt` payloads into next-skill, defer, owner-target,
status-posture, and reason-code counts for the observed window.

The authored meaning lives in
`stats/read-models/active/session_growth_branch_summary.profile.json`; the
stable public contract and output remain
`schemas/session-growth-branch-summary.schema.json` and
`generated/session_growth_branch_summary.min.json`. Deterministic projection
is shared through `src/aoa_stats_builder/growth_cycle.py`.
